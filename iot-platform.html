平台功能

支持多集团公司、多机房、多用户独立权限；全机房全设备监控；全自动控制；3D组态

  • 设备管理

    设备分组、设备创建与修改删除、设备信号维护，设备通信方式配置

  • 设备综合监控

    使用频率非常高的功能模块，可以在这里以树形查看设备分组与各个设备及设备信号的最新状态值，点击不同的设备信号时，可以直接查看当前信号的历史数据更表和曲线

  • 设备控制参数配置

    可配置系统使用手动模式、全自动模式运行；以及配置设备的运行模式和阈值等

  • 手动模式

    手动控制设备启停，切换空调以原模式或节能模式运行

  • 全自动模式

    平台自动为水冷机组、新风系统、精密空调、喷雾系统做实时智能调控；基于大数据计算冷量需求，PID计算水泵水塔及空调的运行频率

  • 3D组态

    基于Blender + ThreeJS实现的3D设备组态，实时显示按楼层或按机组等方式的设备状态

  • 自助报表

    系统内置如用电量报表、PUE报表、COP/SCOP报表等，同时也开放自助配置分析报表，只需选择需要分析的信号数据及图表类型即可轻松实现

技术栈

Golang、Vuejs、ElementUI、PG+时序库、Redis、Docker、Modbus、Mqtt

  • 云网关

    基于ModbusTCP/RTU、MQTT等通信方式与边缘网关、DTU、串口服务器等设备集成，完成数据采集、数据清洗、告警检测、数据入库等完整流程

  • API

    支撑系统的所有功能，不论是北京项目、深圳项目、广州项目，也不论是施工项目或是合同能源管理，一套系统每个账号密码即可管理权限内的所有项目，系统支持纯监控，同时也支持全自动的设备控制

  • 多平台

    支持在WIN/MACOS/LINUX等PC上使用，同时支持平板电脑和手机

  • 历史数据永久保存

    永久保存历史数据，可随时查看任何时间节点的设备历史运行参数及状态，有效为全自动智能调控提供依据

  • 数据自动备份

    全自动备份数据，可以使用备份把系统还原到任何指定的时间节点
